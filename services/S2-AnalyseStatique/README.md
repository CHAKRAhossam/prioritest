# Service 2 - Analyse Statique

**Responsable :** Haytam Ta  
**Email :** haytamnajam14@gmail.com

## Description

Extraction de métriques de code : LOC, complexité cyclomatique (McCabe), CK (WMC, DIT, NOC, CBO, RFC, LCOM), dépendances, smells.

## Technologies

- JavaParser pour Java
- radon pour Python
- CK (Chidamber & Kemerer)
- PMD pour code smells
- SonarQube API
- JGraphT / networkx pour dépendances
- Feast (feature store)

## User Stories

- US-S2-01: Extraction métriques Java
- US-S2-02: Extraction métriques Python
- US-S2-03: Analyse des dépendances
- US-S2-04: Feature Store avec Feast
- US-S2-05: Support multi-projets
